# Bioinformatics – Lunar Lander (Deep Reinforcement Learning)

## 📌 Overview
This is a **Deep Reinforcement Learning (DRL)** solution for the Lunar Lander environment. The agent is trained using a **Deep Q-Network (DQN)** to learn optimal landing strategies through interaction with the environment.

I used the strucutre from https://github.com/mengfn/COMP532-Machine-Learning-and-Bioinspired-Optimisation/blob/main/code/lunar_lander_starter.ipynb ~ my professor Uni of Liverpool
